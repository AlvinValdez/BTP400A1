To run the program within the command-line execute the command below. Make sure you are in the directory of the jar file.
java -jar avvaldez_btp400a1.jar

To extract the jar file you execute the command below. Make sure you are in the directory of the jar file.
jar xf avvaldez_btp400a1.jar
